# C-V2XInteroperabilityTestingTool

This repository contains a set of software testing tools that include automated Cellular Vehicle-to-Everything (C-V2X) interoperability testing procedures developed using Python script. These software testing tools can be used to automatically test C-V2X communication and interoperability among commercial on-board units (OBUs) and road-side units (RSUs) based on IEEE 1609.2, IEEE 1609.3, and SAE J2735 standards to achieve and assure devices interoperability.
